# Sentinel Ops Console — Security Systems KPI Dashboard (Concept)

A sample operations dashboard designed to demonstrate UX and data visualization thinking for a field service / systems monitoring context — access control, cameras/NVR, and ticket management.

**[View live demo →](https://arid871.github.io/usis-dashboard-demo/)**

## What this is

A self-directed concept piece, built to show how I'd approach dashboard UX for a technical support / systems monitoring workflow. All data shown (sites, tickets, metrics) is fictional and for demonstration purposes only.

## What it demonstrates

- **KPI summary cards** — open tickets, response time, system uptime, and SLA compliance at a glance
- **Prioritized ticket table** — triage view with priority, status, and assignment
- **Live site status wall** — at-a-glance health monitoring across multiple locations
- **Weekly resolution trend** — quick visual of throughput over time

## Design approach

The visual language (dark control-room palette, monospace data readouts, status-color coding) is intentionally modeled on real-world monitoring/NOC interfaces, prioritizing quick scannability and clear status signaling over decoration.

## Built with

HTML/CSS — no frameworks, single self-contained file.
